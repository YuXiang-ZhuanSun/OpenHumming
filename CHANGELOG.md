# Changelog

## v1.0.0

- Promoted OpenHumming into a release-ready local-first agent product with a rewritten homepage README and showcase assets.
- Added stronger memory evolution so newer collaboration preferences can replace stale entries in `agent.md` and `user.md`.
- Added reuse-aware skill learning so repeated successful workflows refresh existing drafts with `times_reused` evidence.
- Extended daily review so it consolidates real activity into memory and promotes stronger skill drafts.
- Added a release kit with release notes, PR copy, demo script, and launch posts.
- Added a real demo suite and evolution showcase that demonstrate the product loop end to end.

## v0.9.0

- Added per-turn memory proposals with safe writes back into `agent.md` and `user.md`.
- Added workflow capture, skill draft evaluation, and automatic draft persistence.
- Added daily review promotion for skill drafts plus richer review outputs.
- Added a lightweight local `/ui` chat console and evolution showcase endpoint.
- Added bilingual README and documentation polish for the v1.0 release lane.
