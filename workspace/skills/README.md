# Workspace Skills

Put reusable workflow skills here.
把可复用的工作流技能放在这里。

Each skill is a markdown file that explains:
每个 skill 都是一个 Markdown 文件，应该说明：

- what it does
- 它解决什么问题
- when to use it
- 什么时候使用
- what inputs it needs
- 它需要哪些输入
- what steps it follows
- 它遵循哪些步骤
- what output it should generate
- 它最终应该产出什么
